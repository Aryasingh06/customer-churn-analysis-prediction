#  Customer Churn Analysis
## Project Overview
This project focuses on analyzing customer churn behavior in a company using SQL and Python. The goal of this project is to
identify the major factors contributing to customer churn and provide meaningful business insights and recommendations to
improve customer retention.

-The project includes:
- Data Cleaning
- SQL Analysis
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights & Recommendations

## Problem Statement
Customer churn is one of the biggest challenges faced by telecom companies. Losing customers affects revenue and 
business growth. This project analyzes customer data to understand why customers leave the company and which factors
influence churn behavior the most.

## Dataset Used
- IBM Telco Customer Churn Dataset

Dataset contains:
- Customer demographics
- Contract details
- Payment methods
- Internet services
- Monthly charges
- Churn status

## Key Insights
- The telecom company has a churn rate of 26.54%.
- Customers with month-to-month contracts have the highest churn.
- Customers with higher monthly charges tend to churn more.
- New customers show higher churn behavior.
- Electronic check users have higher churn rates.
- Long-term customers show better retention.

## Business Recommendations
- Encourage customers to switch to long-term contracts.
- Provide loyalty rewards for long-term customers.
- Improve onboarding experience for new customers.
- Review pricing strategies for high-charge customers.
- Develop targeted retention campaigns for high-risk customers.
